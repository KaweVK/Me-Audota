## 📖 Sobre o projeto

O **MeAudota** é uma plataforma criada para facilitar a divulgação de animais disponíveis para adoção. A ideia nasceu da necessidade de centralizar informações que normalmente ficam espalhadas em redes sociais, grupos ou publicações isoladas, dificultando o encontro entre quem deseja adotar e quem está cuidando de um pet resgatado.

A proposta é oferecer um ambiente digital onde protetores independentes, ONGs e responsáveis possam cadastrar pets com informações relevantes, enquanto interessados em adoção conseguem navegar, filtrar e encontrar um novo companheiro com mais facilidade.

Além de ser um projeto tecnológico, o **MeAudota** também busca gerar impacto social, incentivando a **adoção responsável** e ampliando a visibilidade de animais que precisam de um lar.

---

## 🎯 Objetivos

- Facilitar a divulgação de pets disponíveis para adoção;
- Centralizar informações em uma única plataforma;
- Aproximar adotantes de protetores e instituições;
- Tornar a busca por adoção mais prática e organizada;
- Incentivar a adoção consciente e responsável.

## ✨ Funcionalidades

### Funcionalidades principais
- Cadastro de pets para adoção;
- Listagem de animais disponíveis;
- Página de detalhes de cada pet;
- Exibição de informações como:
  - nome;
  - idade;
  - porte;
  - sexo;
  - localização;
  - descrição;
  - fotos;
- Contato com o responsável pela adoção.

### Funcionalidades futuras
- Sistema de login e autenticação;
- Área administrativa para gerenciamento de cadastros;
- Busca e filtros avançados;
- Favoritos;
- Formulário de intenção de adoção;
- Status do pet:
  - disponível;
  - em processo de adoção;
  - adotado;
- Integração com mapas para localização.

## 🚀 Como executar o projeto

### Pré-requisitos

Antes de começar, você vai precisar ter instalado na sua máquina:

- Git
- Node.js
- Java 21
- PostgreSQL

### Clone o repositório
```
git clone https://github.com/seu-usuario/meaudota.git
cd meaudota
```

### Rodando o frontend
```
cd frontend
npm install
npm run dev
```

### Rodando o backend
```
cd backend
./mvnw spring-boot:run
```

No Windows
```
mvnw spring-boot:run
```
